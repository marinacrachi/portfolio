PLAGUE0 - Socket.IO/NodeJS (2018)

Plague0 is a escape room with digital interaction, and the main objective of the game is to develop the teamwork of the players. They got confined in a room and, interacting with a virtual character, solved some puzzles.

My main role in this project was to develop the back-end using Socket.Io/NodeJS, validating the games. I also helped with some ideas for the room, the team dynamics and created custom profiles for each player using the information that the customer gave to me.

For me this project not only improved my technical skills but my soft skills have grown a lot. I learned a lot about game development, teamwork and how to deal with pressure.

BRPF 2017 - Ionic/NodeJS/MongoDB (2017)

BRPF is the national Pump it Up championship, and this app was made for players. The app updated the player scores for each category (Pro Male, Pro Female, Begginers) and also had the stages.

My main role in this project was to develop the back-end using Socket.Io/NodeJS, validating the games. I also helped with some ideas for the room, the team dynamics and created custom profiles for each player using the information that the customer gave to me.

For me this project not only improved my technical skills but my soft skills have grown a lot. I learned a lot about game development, teamwork and how to deal with pressure.
